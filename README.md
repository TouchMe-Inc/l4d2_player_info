# About player_info
Plugin displays information about players (County, lerp, hours).

![image](https://github.com/TouchMe-Inc/l4d2_player_info/assets/89782512/fab4eeda-1f7d-4a69-ba01-4ae31357e38c)

## Commands
* `!info` - Display player info.

## Require
* Colors
* [SteamWorks](https://github.com/hexa-core-eu/SteamWorks)
