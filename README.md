# About player_info
Plugin displays information about players (County, lerp, hours).

## Commands
* `!info` - Show player list.
* `!info <name>` - Display player info.

## Require
* Colors
* Geoip
* [SteamWorks](https://github.com/hexa-core-eu/SteamWorks)
