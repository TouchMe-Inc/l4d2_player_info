# About player_info
Plugin displays information about players (lerp, hours).

![image](https://github.com/TouchMe-Inc/l4d2_player_info/assets/89782512/9ae046ac-d644-4491-9b24-a98f68cb28b8)

## Commands
* `!info` - Display player info.

## Require
* Colors
* [SteamWorks](https://github.com/hexa-core-eu/SteamWorks)
